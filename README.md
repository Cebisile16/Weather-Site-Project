# 🌤️ Weather Forecast Site

A simple and responsive weather forecast web page built with **HTML**, **CSS**, and **JavaScript**. The application displays the current weather for Cape Town along with a three-day forecast. Users can also change the displayed city using a prompt.

## 📖 Overview

This project demonstrates the fundamentals of front-end web development by combining:

* Semantic HTML
* CSS styling and hover effects
* Basic JavaScript DOM manipulation
* User interaction through prompts

Although the weather information is currently static, the city name updates dynamically when a user enters a new location.

## ✨ Features

* Displays the current weather for Cape Town
* Three-day weather forecast
* Responsive weather card layout
* Interactive **Change City** button
* Smooth hover effects
* Clean and modern user interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla)

## 🚀 Getting Started

### Prerequisites

All you need is a modern web browser such as:

* Google Chrome
* Microsoft Edge
* Mozilla Firefox
* Safari

### Running the Project

1. Clone the repository.

```bash
git clone https://github.com/Cebisile16/weather-forecast.git
```

2. Navigate into the project folder.

```bash
cd weather-forecast
```

3. Open `index.html` in your preferred web browser.

No installation or additional dependencies are required.

## 🎮 How to Use

1. Open the webpage.
2. View the current weather and forecast.
3. Click the **Change City** button.
4. Enter the name of any city.
5. The heading updates to display the new city name.

> Note: The weather temperature remains static. Only the city name changes.
